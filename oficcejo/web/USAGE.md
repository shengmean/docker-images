# 使用指南

## 系统概览

本系统是一个基于通达信数据源的股票数据可视化平台，提供实时行情查询和历史数据分析功能。

## 数据获取能力

### 1. 实时行情数据
- **五档报价**：买一到买五、卖一到卖五的价格和挂单量
- **基本指标**：最新价、涨跌幅、成交量、成交额、换手率等
- **当日数据**：开盘价、最高价、最低价、昨收价

### 2. K线数据（历史价格走势）
- **分钟级别**：1分钟、5分钟、15分钟、30分钟、60分钟
- **日级别**：日K线、周K线、月K线、季K线、年K线
- **数据内容**：开盘价、收盘价、最高价、最低价、成交量、成交额

### 3. 分时数据
- **时间维度**：交易日内每分钟的价格和成交量
- **数据范围**：9:30-11:30、13:00-15:00
- **可视化**：分时价格曲线和成交量柱状图

### 4. 分时成交（逐笔交易）
- **成交时间**：精确到秒的成交时间
- **成交价格**：每笔交易的成交价格
- **成交量**：每笔交易的成交数量
- **买卖性质**：标识主动买入或主动卖出

### 5. 股票列表
- **市场范围**：上海证券交易所、深圳证券交易所、北京证券交易所
- **品种类型**：A股、ETF、指数等
- **搜索功能**：支持按代码或名称搜索

### 操作步骤

#### 步骤1：搜索股票

**方法一：直接输入代码**
```
输入：000001
结果：直接显示平安银行的数据
```

**方法二：模糊搜索**
```
输入：平安
结果：显示包含"平安"的所有股票列表
       - 000001 平安银行
       - 601318 中国平安
       ...
点击选择目标股票
```

#### 步骤2：查看五档行情

点击"五档行情"标签，显示：

```
卖五  12.55  1000手
卖四  12.54   800手
卖三  12.53   650手
卖二  12.52   500手
卖一  12.51   300手
─────────────────
买一  12.50   350手
买二  12.49   520手
买三  12.48   700手
买四  12.47   850手
买五  12.46  1100手
```

**数据解读**：
- 左列：档位名称
- 中列：价格（红色为卖盘，绿色为买盘）
- 右列：挂单量（1手=100股）

#### 步骤3：查看K线图

点击"K线图"标签，然后选择周期：

**日K线** - 查看每日价格走势
- 适用场景：中长期投资分析
- 数据范围：最近800个交易日

**30分钟K线** - 查看30分钟级别走势
- 适用场景：短期波段操作
- 数据范围：最近数月的30分钟数据

**5分钟K线** - 查看5分钟级别走势
- 适用场景：日内交易分析
- 数据范围：最近数周的5分钟数据

**图表操作**：
- 鼠标滚轮：缩放图表
- 拖动下方滑块：调整显示范围
- 鼠标悬停：查看具体数据

#### 步骤4：查看分时图

点击"分时图"标签，显示当日分时走势：

- **蓝色曲线**：价格走势
- **蓝色柱状图**：分时成交量
- **时间轴**：9:30-11:30、13:00-15:00

**适用场景**：
- 观察当日价格波动
- 判断买卖时机
- 分析成交量变化

#### 步骤5：查看分时成交

点击"分时成交"标签，显示逐笔交易：

```
时间      价格    成交量   成交额    性质
14:59:58  12.50   100手    12.5万   买入
14:59:55  12.49    50手     6.2万   卖出
14:59:52  12.50   200手    25.0万   买入
...
```

**数据解读**：
- 红色：主动买入（买方主动成交）
- 绿色：主动卖出（卖方主动成交）
- 成交量：以手为单位（1手=100股）

```
目的：分析主力资金的买卖行为

4. 观察集中成交时段
```

## 技巧与建议

### 搜索技巧
1. 使用股票代码搜索最快（6位数字）
2. 记不清代码时，用股票名称关键词搜索
3. 深市股票代码通常以000、002、300开头
4. 沪市股票代码通常以600、601、603开头

### 数据解读技巧
1. **涨跌幅** > 3%：波动较大，注意风险
2. **成交量**放大：市场关注度提升
3. **买一卖一价差**：价差越小，流动性越好
4. **分时成交**：大单频繁出现可能有主力操作

### 分析方法建议
1. **多周期结合**：同时观察日K、30分钟K、5分钟K
2. **量价配合**：价格上涨时成交量应同步放大
3. **盘口观察**：买盘强于卖盘时，价格易上涨
4. **时间节点**：开盘、收盘前后是重要观察时段

## 常见问题解答

### Q1: 数据更新频率是多少？
A: 实时行情数据约3-5秒更新一次，历史数据为交易日结束后更新。

### Q2: 为什么有些股票搜索不到？
A: 系统主要支持A股市场，暂不支持港股、美股等境外市场。

### Q3: K线图显示不完整怎么办？
A: 可以通过底部滑块或鼠标滚轮调整显示范围。

### Q4: 分时数据为什么有断点？
A: 11:30-13:00是午休时间，没有交易，所以会有断点。

### Q5: 如何判断数据是否准确？
A: 可以和其他行情软件对比验证，通达信数据来源可靠。
